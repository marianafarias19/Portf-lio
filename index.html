Meus Projetos

PetLife

Este site apresenta uma clínica veterinária que oferece cuidados especializados para pets, incluindo serviços de saúde, vacinação, cirurgias e atendimento de emergência. Além disso, o site conta com uma seção de petshop e farmácia, oferecendo produtos de bem-estar e cuidados para animais. Com um layout moderno e responsivo, a plataforma permite aos usuários agendar consultas via WhatsApp, explorar os serviços disponíveis e tirar dúvidas sobre os cuidados com os pets. A navegação é intuitiva, com seções claras e informações detalhadas para facilitar a experiência dos donos de animais.

To-Do-List

O To-Do List é um aplicativo simples e funcional que permite aos usuários gerenciar suas tarefas diárias. Ele possibilita adicionar novas tarefas, marcar tarefas como concluídas e excluir tarefas da lista. As tarefas concluídas são automaticamente destacadas com uma linha através do texto. A interface é intuitiva, permitindo uma navegação fácil, e o aplicativo é responsivo, adaptando-se a diferentes dispositivos.

Reserva de Hotel

Este projeto é uma criação de um site de reserva de hotel, desenvolvido com HTML, CSS e JavaScript. A página apresenta um layout moderno e responsivo, com seções para seleção de datas, e exibição de opções de acomodação. O design utiliza Flexbox e Grid para garantir adaptação em diferentes dispositivos. A funcionalidade interativa, como a escolha de datas e o filtro de resultados, é implementada com JavaScript, proporcionando uma experiência de usuário intuitiva. O objetivo é simular a aparência e as funcionalidades básicas de um site de reservas, com ênfase na usabilidade e design limpo.
